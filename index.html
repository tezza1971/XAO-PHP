Welcome to XAO

Introduction

WELCOME TO XML APPLICATION OBJECTS version 1.0alpha. XAO is a suite of PHP classes which [web] application developers can use to speed up application development. XAO is also a scalable framework designed to be extended by the developer. XAO is implemented using standards based technology as specified by the W3C. It utilises XML based technologies, namely the Document Object Model API and XSL-Transformations (XSLT).

About this release (1.0a)

XAO is (and always will be) a work in progress. There are actually many other classes under development which are not included in this distribution. We have big plans for the future. In the mean time, what you have here is the core set of functionality. Which, as it happens, is enough to build scalable, well-designed, object oriented, standards compliant web applications.

This is an Alpha release, so please report any bugs or strange behavior to our discussion forum or the mailing list. In fact, we would be grateful for any feedback at all, so by all means, stop by the forum for a chat or register on the mailing list.

Install notes

After meeting the system requirements as mentioned below, it will pay to copy the contents of the "classes" directory of this distribution into your PHP includes directory. For more information on where your PHP include directory is, see the include_path directive in your PHP.ini file. You may like to simply add an additional path which points to the classes directory of this distribution.

The alternative to locating the classes in the include directory is to provide comprehensive path information every time you include one of the library files (not good). You'll note how awkward this is when you see the source-code in the tutorials. The tutorials were done this way so they would work out-of-the-box.
